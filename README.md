# Отказы Б24

AI-pipeline для анализа проигранных сделок (стадия «Отказ») в Bitrix24.
Для каждой сделки определяет: был ли отказ корректным и можно ли вернуть клиента.

Стек: n8n + Bitrix24 REST API + Yandex SpeechKit + Polza.ai (Claude 3.7 Sonnet) + Supabase.
